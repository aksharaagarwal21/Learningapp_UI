# SkillForge — Skill Assessment & Learning Platform

A modern, production-grade frontend for a skill assessment and learning platform built with **React 19**, **TypeScript**, **Vite**, **Framer Motion**, and **Recharts**.

## Pages

| Page | Route | Description |
|------|-------|-------------|
| Landing | `/` | Hero section, features, testimonials, pricing, footer |
| Login | `/login` | Email login + quick demo login for 3 roles |
| Signup | `/signup` | Multi-step registration with role selection |
| Dashboard | `/dashboard` | Stats, assessments, activity heatmap, recommendations |
| Courses | `/courses` | Browse all courses with progress tracking |
| Course Detail | `/courses/:id` | Overview, curriculum, prerequisites, start/continue |
| Course Learn | `/courses/:id/learn` | Video, reading, code editor, quizzes with sidebar nav |
| Assessments | `/skill-assessment` | Ongoing + filterable assessment catalog |
| Assessment Detail | `/assessment/:id` | Rules, topics, and start card |
| Assessment Take | `/assessment/:id/take` | Timed quiz with question navigation and flagging |
| Assessment Results | `/assessment/:id/results` | Score ring, pass/fail, question-by-question review |
| Certificates | `/certificates` | Earned certificates gallery |
| Leaderboard | `/leaderboard` | Animated podium + sortable ranking table |
| Practice | `/practice` | Daily challenges and topic-based practice |
| Profile | `/profile` | Skill radar chart, badges, completion tracker |
| Settings | `/settings` | Theme, notifications, security, account |
| Notifications | `/notifications` | Categorized notification feed |
| Recruiter Dashboard | `/recruiter/dashboard` | Job posts, candidates, skill demand chart |
| Instructor Dashboard | `/instructor/dashboard` | Performance trends, assessments, feedback |

## Tech Stack

- **React 19** + **TypeScript** + **Vite 8**
- **Framer Motion** — animations & transitions
- **Recharts** — radar, bar, area charts
- **Lucide React** — icons
- **CSS Custom Properties** — light/dark theme
- **Plus Jakarta Sans** — typography

## Run Locally

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`. Use the **Quick Demo Login** buttons on the login page to test each role.
